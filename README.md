# Local Issue Reporting System

A modern web-based platform that enables citizens to report local community issues such as potholes, garbage accumulation, water leakage, streetlight failures, and other public concerns. The system allows users to submit complaints, track their status, and stay informed throughout the resolution process, while administrators can efficiently manage and monitor reported issues.

## Features

### User Module

* User Registration and Login
* Report Local Issues
* Upload Supporting Images
* Track Complaint Status
* View Complaint History
* Responsive Dashboard

### Admin Module

* Secure Admin Login
* Dashboard Overview
* Complaint Management
* Update Complaint Status
* User Management
* Analytics and Reports

### Complaint Lifecycle

1. Complaint Submitted
2. Complaint Verified
3. In Progress
4. Resolved

## Installation

### Clone Repository

```bash
git clone https://github.com/your-username/local-issue-reporting-system.git
```

### Navigate to Project Directory

```bash
cd local-issue-reporting-system
```

### Install Dependencies

```bash
npm install
```

### Start Development Server

```bash
npm run dev
```

## Purpose

The Local Issue Reporting System aims to bridge the gap between citizens and local authorities by providing a transparent and efficient platform for reporting and resolving community issues. The project promotes civic engagement, accountability, and improved public service delivery.
